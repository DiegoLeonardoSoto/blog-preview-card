### What I learned

I learned how to integrate Tailwind CSS into a basic HTML/CSS project and how to connect and use MCPs within OpenCode.

package.json
```json
"scripts": {
  "build:styles": "pnpx @tailwindcss/cli -i ./input.css -o ./assets/output.css --watch"
}
```

input.css

```css
@import "tailwindcss";
```

```json
{
  "files.associations": {
    "*.css": "tailwindcss"
  }
}
```
